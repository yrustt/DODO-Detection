Используется python3.10 и poetry 2.2.1.

Запуск:

1. poetry install
2. poetry run python main.py --video=path/to/video.mp4

Результат в файле output.mp4 и в консоле.

Какая логика использовалась для детекции событий:

1. Обнаружение и фиксация столов, чтобы избежать того, что столы то появляются то ищезают.
2. Детекция движения людей, чтобы исключить моменты, когда человек проходит мимо стола.

Средний простой в первые 1.5 минуты - 166 кадров.

Пример обработанного видео доступен по [ссылке](https://disk.yandex.ru/i/4PaHceggWhG5dw).
